# Ejecutar proyecto en desarrollo

1. Clonar el repositorio

## SSH

```
git@github.com:KevinDanielDev/NextTest.git
```


## HTTPS

```
https://github.com/KevinDanielDev/NextTest.git
```


3. Instalar dependencias
```
npm install || npm i
```

4. Correr proyecto
```
npm run dev
```

5. Entrar al navegador
```
localhost:3000
```
